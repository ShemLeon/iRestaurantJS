# DropDownData

## Описание

Модуль для хранения данных, используемых в меню навигации и выпадающих меню (dropdown).

## Функциональность

### Хранение данных

- Хранит структуры данных для меню и dropdown
- Содержит конфигурацию элементов навигации

### Работа с изображениями

- Импортирует необходимые изображения для элементов меню
- Управляет путями к графическим ресурсам

### Экспорт данных

- Экспортирует структуры данных для использования компонентами
- Предоставляет централизованный доступ к данным меню
