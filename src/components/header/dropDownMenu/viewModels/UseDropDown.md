# useDropDown Hook

## Описание

Хук для управления состоянием открытия/закрытия выпадающих меню (dropdown) в навигационной панели.

## Как работает

### Состояние

- `useState(null)` — хранит имя открытого dropdown (или `null`, если ни один не открыт)

### Функции

- **`handleMouseEnter(itemName)`** — открывает dropdown для указанного `itemName`
- **`handleMouseLeave()`** — закрывает текущий открытый dropdown
- **`isDropdownOpen(itemName)`** — проверяет, открыт ли dropdown для указанного `itemName`

## Возвращаемое значение

Хук возвращает объект, содержащий:
- Состояние открытого dropdown
- Функции для управления открытием/закрытием
